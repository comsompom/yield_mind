How to use YieldMind

YieldMind helps you analyze opportunities, execute operations, and track actions in one flow. This page summarizes what you can achieve and how to use each function.

What you can achieve

  • Compare wallet balances and best opportunities by APY and risk.
  • Bridge assets into Initia and send tokens directly from the app.
  • Get AI recommendations with reasoning and trend context.
  • Track your operations in transaction history for demo and review.

Quick start

  1. Connect your Initia-compatible wallet in the top-right corner.
  2. Open Dashboard to review balances and top opportunities.
  3. Use Operations for Send, Bridge, and transaction history.
  4. Use AI Recommendations to ask strategy questions and execute ideas.

Functions summary

Page / Function What it does What user can do
Dashboard Portfolio overview and APY comparison chart. Identify best opportunities quickly.
Operations / Send Transfer INIT to any valid address. Input recipient + amount, sign, get tx hash.
Operations / Bridge Prepare bridge-in flow from source chain to Initia. Preview route, submit bridge, track status.
Operations / Transaction history Recent actions list with status and explorer links. Verify each operation and present proof.
AI Recommendations Prompt-based suggestions with prediction chart support. Request strategy, inspect reasoning trend, execute suggestion.
