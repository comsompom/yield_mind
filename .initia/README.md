# Initia hackathon submission

- **submission.json** — Fill in before submitting: `repo_url`, `live_app_url`, `demo_video_url`, `chain_id`, `team`. Set `chain_id` to your **saved rollup chain ID** (from `weave init`, e.g. `yieldmind-1`). See [get-started docs](https://docs.initia.xyz/hackathon/get-started).
- **local-ids.example.md** — Copy to `local-ids.md` and fill in as you run `weave init` (Gas Station address, rollup chain ID). Do not commit `local-ids.md` (it is in `.gitignore`).
